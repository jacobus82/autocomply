How to add this signature to your email client
📧
Gmail
  1. Click Settings (⚙) → See all settings
  2. Go to the General tab
  3. Scroll to Signature → click Create new
  4. Click the <> (source) icon in the editor toolbar
  5. Paste the copied HTML and click OK
  6. Scroll down and click Save Changes
📨
Outlook (Desktop)
  1. Go to File → Options → Mail → Signatures
  2. Click New and name your signature
  3. In the editor, click the Source button (or paste the plain text version for simplicity)
  4. Paste the HTML and save
  5. Set it as your default for new messages under Choose default signature

Tip: Outlook's HTML support is limited — the Professional or Clean style works most reliably.

🍎
Apple Mail
  1. Open Mail → Preferences (⌘,) → Signatures tab
  2. Select your email account, click + to create a new signature
  3. Paste the HTML into the signature area (or use plain text version)
  4. Uncheck Always match my default message font

Tip: For best Apple Mail results, paste the HTML into a text editor, save as .html, then drag the file into the signature panel.

📱
Mobile / other clients
Most mobile email apps don't support HTML signatures — use the plain text version for mobile. For Gmail app: Settings → your account → Signature settings.
